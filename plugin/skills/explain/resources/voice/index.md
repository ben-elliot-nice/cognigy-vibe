## voice — Voice Overview

Covers Voice Gateway–specific concerns: endpoint routing and session config for the VG channel
(`voice-gateway`), and the silence-detection/reprompt-then-escalate pattern for `noUserInput`
handling (`voice-silence-timeout`).

Reach for this group whenever a flow targets the VoiceGateway channel specifically, as opposed to
digital/chat channels.
